## How to run the code
- git clone 
- cd laravel-starter
- cp .env.example `.env`
- open .env and update DB_DATABASE (database details)
- `composer install`
- `php artisan key:generate`
- `php artisan migrate:fresh --seed`
- `php artisan serve`
